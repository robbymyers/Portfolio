Problem — Solution

From Friction to Flow

How we translated fragmented user pain points into a unified, streamlined product experience.

"How might we help athletes track both past and upcoming events across multiple fitness modalities — so they can reflect on their journey while planning future goals — while simplifying event sign-up into a seamless experience?"
Friction Fragmented Discovery Athletes used 3–5 separate platforms to find events across running, cycling, and CrossFit. 4 of 6 competitors were modality-specific.
→
Solution Unified multi-sport search with location-based filtering and modality toggles in a single interface.
Friction 4-Step Registration Event registration required navigating away from the discovery page, creating accounts on external sites, and filling redundant forms.
→
Solution Inline 2-step registration flow — select event, confirm details. No context switching.
Friction No Performance Tracking Athletes couldn't see past race times or plan future goals in one place. Data was scattered across race-specific websites.
→
Solution Unified profile with chronological event history, split times, and modality-filtered performance views.
Friction Organizer Fragmentation Event organizers had to manage registrations across multiple URLs and platforms for each event they hosted.
→
Solution Centralized organizing dashboard for creating, managing, and tracking participants across all events.
